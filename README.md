## About

A highly interactive 3D portfolio website featuring animated scenes, smooth scroll-based transitions, project showcases, and a working contact form. Built to stand out and make a lasting impression.

## Live Demo

[View Portfolio](https://abdullah-naseem-abbasi-portfolio.vercel.app)

## Tech Stack

- **React 19** — UI Framework
- **Three.js / React Three Fiber / Drei** — 3D Graphics & Interactive Models
- **GSAP** — Scroll-based & Reveal Animations
- **Tailwind CSS v4** — Utility-first Styling
- **EmailJS** — Contact Form Integration
- **Vite 6** — Build Tool & Dev Server

## Features

- Interactive 3D hero section with animated room model
- Smooth GSAP scroll-triggered animations
- Project showcase with live demo & GitHub links
- Professional experience timeline
- 3D tech stack cards
- Client testimonials (masonry grid)
- Fully responsive design (mobile, tablet, desktop)
- Working contact form with email delivery
- Mouse-tracking glow card effects

## Quick Start

```bash
# Clone the repo
git clone https://github.com/AbdullahNaseemAbbasi/abdullah-naseem-abbasi-portfolio.git
cd abdullah-naseem-abbasi-portfolio

# Install dependencies
npm install

# Create .env file
cp .env.example .env
# Fill in your EmailJS credentials (see .env.example)

# Start dev server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Environment Variables

| Variable | Description |
|----------|-------------|
| `VITE_APP_EMAILJS_SERVICE_ID` | EmailJS Service ID |
| `VITE_APP_EMAILJS_TEMPLATE_ID` | EmailJS Template ID |
| `VITE_APP_EMAILJS_PUBLIC_KEY` | EmailJS Public Key |
